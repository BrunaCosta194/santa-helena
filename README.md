# Sacolão Santa Helena 🥬

Pedidos online para um sacolão/hortifruti ("iFood do mercado"). A dona publica o que tem
hoje e as promoções; o cliente pede pelo site. Vende frutas, verduras, temperos, polpas e
saladas prontas — com venda por kg e por unidade. Pagamento na entrega (online na fase 2),
login opcional com base de clientes, e aviso de pedido no WhatsApp.

*"Qualidade · Frescor · Economia"*
